Graphics

This section is dedicated to show some personal projects applying different techniques of image treatment, montage, color composition and other concepts which are usually applied to the graphic design segment but can also be applied in other game elements. The prupose of these works is to show the knowlede of some basic concepts of graphic design.


Basic Painting

Basic painting work using light source, shadowing and color palete.


Composition

Simple poster example using color matching, image composition, perspective, shadow applying, Z reading concept and light source illumination.



Retouching

Retouching technique with noise reduction, Image treatment, RGB levels alignment and filter applying.



Shadowing

Inner shadow drawing, casting shadow, illumination and filters applied to create realistic composition.



Montage

Scene composition, character creation, color matching and montage applied to create a character and a matching background.
